# Basic HTML Website

## Overview

This is a multi-page website built using only HTML5 as part of my web development learning journey. The project focuses on creating a well-structured website using semantic HTML elements without CSS or JavaScript.

## Features

- Multi-page website
- Semantic HTML structure
- Navigation bar on every page
- Homepage
- Projects page
- Articles page
- Contact page
- Contact form
- SEO meta tags
- Responsive viewport meta tag
